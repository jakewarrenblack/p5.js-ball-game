Note:

This game must be run using a local server such as live server for VSCode:
https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer

Alternatively, the game is also hosted on GitHub Pages. You can play it here:
https://jakewarrenblack.github.io/p5.js-ball-game/

You can view the repository itself here:
https://github.com/jakewarrenblack/p5.js-ball-game

The game won't run if you just open the html file. The browser's CORS policy prevents local files from loading.
